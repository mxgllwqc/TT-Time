# TT Clock 

## ✨ Features

- Real-time clock (24h format via system/browser locale)
- Date display: `DD.MM.YYYY | full weekday`
- Drag & drop positioning (saved in localStorage)
- Color toggle (double click)
- Dynamic text outline (black/white depending on color)
- Persistent settings (position + color)
- Reset position (R key or right click)
